CXCursorVisitor デリゲート

Visitor invoked for each cursor found by a traversal. This visitor function will be invoked for each cursor found by clang_visitCursorChildren(). Its first argument is the cursor being visited, its second argument is the parent visitor for that cursor, and its third argument is the client data provided to clang_visitCursorChildren(). The visitor should return one of the CXChildVisitResult values to direct clang_visitCursorChildren().
